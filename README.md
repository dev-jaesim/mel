# Youtube Clone

Youtube Clone project by using MERN stack (MongoDB, ExpressJS, ReactJS and NodeJS)

---

## Features

### Backend

- [x] Login
- [x] Logout
- [x] Registration
- [x] Video upload
- [x] Search
- [x] Video list
- [x] Subscription
- [x] Load more
- [x] Reply
- [x] Like Dislike

### Frontend

- [x] Login
- [x] Logout
- [x] Registration
- [x] Video upload
- [x] Search
- [x] Video list
- [x] Subscription
- [x] Load more
- [x] Reply
- [x] Like Dislike
